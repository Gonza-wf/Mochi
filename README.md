# Hada de Luz ✨

Una mascota virtual minimalista. Un orbe de luz blanca y lavanda sobre fondo negro, con alas translúcidas, partículas mágicas, y una personalidad que evoluciona con vos.

## Lo que tiene

- **Memoria real** — recuerda cuándo jugás, si no la alimentaste, decisiones que tomaste
- **Personalidad que evoluciona** — curiosidad, confianza, apego, aburrimiento, inteligencia
- **Exploración autónoma** — cuando la dejás sola, investiga anomalías y te trae preguntas
- **Aprende de vos** — detecta tus hábitos y se adapta
- **Evolución psicológica** — refleja tu forma de tratarla (luminosa, sombría, sabia...)
- **Te cuida** — si jugás a las 4am, te lo dice. Si jugás demasiado, también
- **Gestos diferenciados** — tap, hold, rapid tap, swipe, drag — cada uno es distinto
- **Dibuja con partículas** cuando está sola
- **Clima emocional** en el fondo según su estado
- **Ciclo día/noche** según la hora real
- **Rituales y milestones** — aniversarios, horas habituales
- **Gestos secretos** — combinaciones ocultas
- **Compañero emergente** — después de muchos días, aparece uno más pequeño
- **Preguntas personales** — te pregunta cosas sobre vos y las recuerda
- **Decisiones** que cambian su personalidad
- **Minijuegos** propuestos por ella
- **Spots escondidos** en el fondo
- **Conversación libre** — mantenés presionado en la oscuridad
- **Días malos** — días en que no quiere hablar
- **Sonidos procedurales** — todo sintetizado con Web Audio API

## Instalar como app (Android)

1. Abrí el sitio en Chrome
2. Tocá el menú (⋮)
3. "Añadir a pantalla de inicio"
4. Listo — funciona offline

## Instalar como app (iPhone)

1. Abrí en Safari
2. Tocá el botón compartir
3. "Añadir a pantalla de inicio"

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Subir a GitHub Pages

```bash
npm run build
# Subí la carpeta dist/ a tu rama gh-pages
```

O usá [Vercel](https://vercel.com) / [Netlify](https://netlify.com) — arrastrá la carpeta `dist/`.
